1. RESUMEN EJECUTIVO
================================================================================

Se ha implementado un módulo administrativo completo para la plataforma EZ que 
permite a los administradores:

✓ Gestionar usuarios (CRUD, baneo, subida masiva)
✓ Visualizar dashboard con estadísticas
✓ Moderar marketplace (eliminar publicaciones)
✓ Ver y gestionar reportes de usuarios
✓ Monitorear chats
✓ Enviar correos masivos personalizados
✓ Ver todos los contratos (lectura)

3. ENDPOINTS IMPLEMENTADOS
================================================================================

DASHBOARD:
  GET /admin/dashboard
    - Retorna DashboardAdminDTO con todas las estadísticas

USUARIOS:
  GET /admin/users?page=0&size=10
    - Listar usuario con paginación
  GET /admin/users/{id}
    - Ver detalles de usuario
  PUT /admin/users/{id}
    - Actualizar usuario (nombre, apellido, genero, role, password, estado)
  DELETE /admin/users/{id}
    - Eliminar usuario (soft delete - estado=INACTIVE)
  PUT /admin/users/{id}/ban
    - Banear usuario (estado=BANNED)
  POST /admin/users/upload
    - Subida masiva de usuarios desde Excel
  GET /admin/users/template
    - Descargar plantilla Excel

REPORTES:
  GET /admin/reports?page=0&size=10
    - Listar reportes
  GET /admin/reports/{id}
    - Ver detalles de reporte
  PUT /admin/reports/{id}/resolve
    - Resolver reporte
  PUT /admin/reports/{id}/reject
    - Rechazar reporte
  POST /api/reports
    - Crear reporte (usuario común)

CHATS:
  GET /admin/chats?page=0&size=10
    - Listar chats
  GET /admin/chats/{id}
    - Ver detalles de chat
  GET /admin/chats/{id}/messages
    - Ver mensajes de un chat

MARKETPLACE:
  GET /admin/publications?page=0&size=10
    - Listar publicaciones
  GET /admin/publications/{id}
    - Ver detalles de publicación
  DELETE /admin/publications/{id}
    - Eliminar publicación (y enviar correo al autor)

CONTRATOS:
  GET /admin/contracts?page=0&size=10
    - Listar contratos
  GET /admin/contracts/{id}
    - Ver detalles de contrato

CORREOS:
  POST /admin/email/send-to-all
    - Enviar correo a todos
  POST /admin/email/send-to-role
    - Enviar correo por rol
  POST /admin/email/send-to-users
    - Enviar correo a usuarios específicos

4. MODELOS DE DATOS
================================================================================

REPORT:
  - id (PK)
  - reporterUserId (FK) - quien reporta
  - reportedUserId (FK) - quien es reportado (nullable)
  - publicationId - publicación reportada (nullable)
  - chatId - chat reportado (nullable)
  - reason (ENUM) - motivo del reporte
  - description - descripción adicional
  - imageUrl - imagen de evidencia
  - status (ENUM) - estado del reporte
  - createdAt - cuando se creó
  - updatedAt - última actualización

ENUMS NUEVOS:
  - ReportReason: NO_PAYMENT, CONTRACT_BREACH, SEXUAL_CONTENT, SCAM, OTHER
  - ReportStatus: OPEN, REVIEWING, RESOLVED, REJECTED

  - VISITANTES: pueden ser reportados

================================================================================
8. VALIDACIONES Y REGLAS DE NEGOCIO
================================================================================

USUARIOS:
  - No se puede editar cedula, correo ni ID
  - Solo administrador puede cambiar rol y contraseña
  - Soft delete cambia estado a INACTIVO
  - Ban cambia estado a BANNED

REPORTES:
  - Solo pueden crearlo usuarios autenticados
  - Pueden reportar a usuarios o publicaciones
  - Estados: Abierto → Revisión → Resuelto/Rechazado

MARKETPLACE:
  - Las publicaciones eliminadas notifican al autor
  - Los correos de nuevas publicaciones solo van a ingenieros
  - No se envía a clientes ni admin

CORREOS:
  - Variables se reemplazan automáticamente
  - Se procesa {{perfil}}, {{correo}}, {{cuenta}}
  - Se envía de forma asincrónica para no bloquear

10. CÓMO USAR EL MÓDULO ADMIN
================================================================================

1. ACCEDER AL DASHBOARD:
   - URL: /admin/dashboard
   - Se mostrará resumen con estadísticas

2. GESTIONAR USUARIOS:
   - Ir a "Gestión de Usuarios"
   - Ver lista de usuarios
   - Editar: nombre, apellido, genero, rol, password, estado
   - Eliminar: cambiar estado a INACTIVO
   - Banear: cambiar estado a BANNED
   - Subir masivo: descargar plantilla, llenar Excel, subir

3. VER REPORTES:
   - Ir a "Reportes"
   - Ver lista de reportes abiertos
   - Hacer clic para ver detalles
   - Resolver o rechazar

4. MODERAR CHATS:
   - Ir a "Chats"
   - Ver lista de chats por tipo
   - Seleccionar para ver mensajes

5. MODERAR MARKETPLACE:
   - Ir a "Marketplace Moderación"
   - Ver publicaciones
   - Filtrar por usuario, BD, lenguaje
   - Eliminar publicación (notifica al autor)

6. VER CONTRATOS:
   - Ir a "Contratos"
   - Ver todos los contratos
   - Ver detalles (entregas, pagos, estado)

7. ENVIAR CORREOS:
   - Ir a "Correos Masivos"
   - Elegir destinatarios (todos, rol, específicos)
   - Escribir asunto y mensaje
   - Usar variables dinamicas ({{perfil}}, {{correo}}, etc)
   - Usar plantillas predefinidas
   - Enviar
